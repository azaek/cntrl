## 0.1.0

Initial release.

- REST and WebSocket API on port 9990.
- Real-time system stats streaming (CPU, memory, GPU, disks, network).
- Media playback detection and control.
- Process listing, kill, launch, and focus.
- Power control (shutdown, restart, sleep, hibernate).
- Topic-based WebSocket subscriptions with hierarchical expansion.
- Optional API key authentication.
- Configurable feature toggles.
- System tray with auto-start support.
